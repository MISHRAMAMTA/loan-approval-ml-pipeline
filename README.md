# Loan Approval Prediction - Machine Learning Pipeline

This project implements an end-to-end ML pipeline for predicting loan approval decisions.

This Project includes 
data preprocessing
feature engineering
model training
evaluation
deployment-ready inference using scikit-learn pipelines and custom transformers.

### Features
- Custom scikit-learn transformers (Drop columns, Log transform, Label encoding)
- Fully automated training pipeline
- Serialized model using joblib for real-world reuse
- Installable Python package (setup.py)
- Prediction script for inference on new data

### Tech Stack
Python, Pandas, NumPy, Scikit-learn, Joblib, Virtualenv, Packaging
